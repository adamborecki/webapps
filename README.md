# webapps
A showcase of web creations like visualizers, kids games, and music tech webapps.

## Music Tech
### MixCube
Imagine a stereo mix as a 3D cube. This is a visualizer to that is loosely inspired by David Gibson's "Art of Mixing":
- https://github.com/adamborecki/mixcube
Version 1: basic 3D mixing
Version 2: includes dynamics processors
